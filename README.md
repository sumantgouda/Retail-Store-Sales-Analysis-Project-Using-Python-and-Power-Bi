# Retail-Store-Sales-Analysis-Project-Using-Python-and-Power-Bi
## Project Overview
This project demonstrates an end-to-end data analytics workflow where Python is used for data cleaning and preprocessing, and Power BI is used to build an interactive sales analytics dashboard. The dashboard provides insights into sales performance, customer behavior, regional trends, product categories, and payment methods.

## Objectives
* Clean and prepare unclean retail sales data using Python
* Perform Exploratory Data Analysis (EDA)
* Build an interactive Power BI dashboard
* Derive business insights for decision-making

## Tools & Technologies Used
Python
* Pandas
* NumPy

Power BI
* Data modeling
* DAX measures
* Interactive visuals

## Data Cleaning (Python)
* Missing values
* Duplicate records
* Inconsistent city & region names
* Invalid ages and outliers
* Incorrect data types
* Formatting issues (extra spaces, casing)
* Derived columns (Age Group, Region, Month)
